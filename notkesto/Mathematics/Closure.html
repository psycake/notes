Closure

Topics: Calculus - Topology

(definition)

We define the closure of a set \(A\) as:

\[\overline A = A \cup A_c(A)\]

‚Ä¶where \(A_{c}(A)\) is the set of all accumulation points of \(A\).

Examples
  • \(\overline{(0,1)} = [0,1]\)
  • \(\overline{\mathbb{Q}} = \mathbb{R}\)
  • \(\overline{\mathbb{I}} = \mathbb{R}\)
  • \(\overline{\{ \frac{1}{n}\ |\ n \in \mathbb{N} \}} = \{ \frac{1}{n}\ |\ n \in \mathbb{N} \} \cup \{0\}\)

Pinglonoto

Topics: Notkesto

A Pinglonoto (Esperanto for "pin note"; pl. Pinglonotoj) is a Gvidonoto that is created just to make it show up in Graph view in a different colour than the default light-grey that Obsidian uses for inexistent notes.

Pinglonotoj can be seen as precursors to both Temomapoj and Kursmapoj. When enough Arbonotoj link to a given Pinglonotoj, it can be promoted to a Temomapo or a Kursmapo.
